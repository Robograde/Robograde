Additional options for uic when using AUTOUIC

This property holds additional command line options which will be used when uic is executed during the build via AUTOUIC, i.e. it is equivalent to the optional OPTIONS argument of the qt4_wrap_ui()<FindQt4> macro.

By default it is empty.

The options set on the .ui source file may override AUTOUIC_OPTIONS set on the target.
